# Basic Bank System - C++

A console-based Bank Management System developed using **C++**.  
This project was built step by step during the C++ course to practice programming fundamentals, file handling, structures, vectors, and user management systems.

The project evolved through multiple extensions, adding more advanced features in each stage.

---

## Project Overview

The Basic Bank System allows users to manage bank clients, perform transactions, and control system access through user permissions.

The application works through a console interface and stores data using text files.

---

# Features

## Course 7 - Basic Bank System

### Client Management
- Show all clients
- Add new clients
- Delete clients
- Update client information
- Find clients using account number

### Transactions
- Deposit money
- Withdraw money
- Show total balances

### Data Storage
- Save clients data into text files
- Load data automatically when the program starts
- Convert data between files and structures

---

## Course 7 Extension

Added more transaction functionality:

- Deposit screen
- Withdraw screen
- Total balances report
- Improved client searching
- Better file management

---

## Course 8 Extension - Users and Permissions

Added a complete user management system.

### Authentication
- Login system
- Username and password verification

### User Management
- Add new users
- Delete users
- Update users
- Find users
- Display users list

### Permissions System
Implemented permission control using binary numbers.

Users can have permissions for:

- Show clients list
- Add clients
- Delete clients
- Update clients
- Find clients
- Transactions
- Manage users

The system supports:
- Full access users
- Limited access users
